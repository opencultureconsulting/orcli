orcli export jsonl

export JSON Lines / newline-delimited JSON

Usage

Arguments
project name or id
(required)
Options
specify if project contains multi-row records
character(s) that separates multiple values in one cell (row mode only)
filter result set by providing an OpenRefine facets config in json
Write to file instead of stdout
set character encoding
suppress log output, print errors only
Parent Environment Variables
URL to OpenRefine server

Examples

orcli export jsonl "duplicates" orcli export jsonl "duplicates" --output "duplicates.jsonl" orcli export jsonl "duplicates" --separator ' ' orcli export jsonl "duplicates" --mode records orcli export jsonl "duplicates" --facets '[ { "type": "text", "columnName": "name", "mode": "regex", "caseSensitive": false, "invert": false, "query": "^Ben" } ]' orcli export jsonl "duplicates" --facets '[{ "type": "list", "expression": "grel:filter([\"gender\",\"purchase\"],cn,cells[cn].value.find(/^F/).length()>0).length()>0", "columnName": "", "selection": [{"v": {"v": true}}] }]'
