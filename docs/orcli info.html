orcli info

show OpenRefine project's metadata

Usage

Arguments
project name or id
(required)
Parent Environment Variables
URL to OpenRefine server

Examples

orcli info "duplicates"
orcli info 1234567890123
orcli info "duplicates" | jq -r .columns[]
