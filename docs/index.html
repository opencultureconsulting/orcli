orcli 0.4.2

OpenRefine command-line interface written in Bash

Commands

  • completions - generate bash completions
  • delete - delete OpenRefine project
  • export - commands to export data from OpenRefine projects to files
    • csv - export comma-separated values (CSV)
    • jsonl - export JSON Lines / newline-delimited JSON
    • template - export to any text format by providing your own GREL template
    • tsv - export tab-separated values (TSV)
  • import - commands to create OpenRefine projects from files or URLs
    • csv - import character-separated values (CSV)
    • json - import JSON
    • jsonl - import JSON Lines / newline-delimited JSON
    • tsv - import tab-separated values (TSV)
  • info - show OpenRefine project's metadata
  • list - list projects on OpenRefine server
  • run - run tmp OpenRefine workspace and execute shell script(s)
  • search - apply regex to each column and print matches in flattened tsv format
  • sort - commands to sort OpenRefine projects
    • columns - re-order columns alphabetically
  • test - run functional tests on tmp OpenRefine workspace
  • transform - apply undo/redo JSON file(s) to an OpenRefine project

Examples

orcli import csv "https://git.io/fj5hF" --projectName "duplicates" orcli list orcli info "duplicates" orcli transform "duplicates" "https://git.io/fj5ju" orcli search "duplicates" "^Ben" orcli sort columns "duplicates" orcli export tsv "duplicates" orcli export tsv "duplicates" --output "duplicates.tsv" orcli delete "duplicates" orcli run --interactive orcli run << EOF orcli import csv "https://git.io/fj5hF" --projectName "duplicates" orcli transform "duplicates" "https://git.io/fj5ju" orcli export tsv "duplicates" EOF
