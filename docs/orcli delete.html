orcli delete

delete OpenRefine project

Usage

Arguments
project name or id
(required)
Options
delete all projects with the same name
suppress log output, print errors only
Parent Environment Variables
URL to OpenRefine server

Examples

orcli delete "duplicates" orcli delete "duplicates" --force orcli delete 1234567890123 for p in $(orcli list); do orcli delete ${p:0:13}; done
